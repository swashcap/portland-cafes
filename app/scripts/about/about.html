Portland Cafés is a fun side project by Portland-based developers and coffee enthusiasts David LaChasse and Cory Reed. The project began as an answer to the question “what coffee shops are open?” and quicky evolved into a complete listing of the city’s many cafés.

The project’s source code is available on GitHub.
